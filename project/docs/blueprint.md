# **App Name**: Melody Grove

## Core Features:

- Track Playback & Controls: Essential play, pause, next, previous track controls, along with a volume slider and scrub bar.
- Album Art Display: Prominently displays the album art for the currently playing soundtrack, using a placeholder image for initial MVP.
- Now Playing Information: Dynamically updates to show the title and artist of the track currently being played, along with current time and total duration.
- Interactive Tracklist: A scrollable list displaying all tracks from the album, allowing users to select any track to play.
- AI-Powered Track Storyteller: An AI tool that generates whimsical, short descriptive snippets or 'mood tags' for each track, enhancing the cute and narrative feel of the game soundtrack, based on its title.

## Style Guidelines:

- Primary color: Soft, pastel pink (#E8C9D3) to evoke a cute and gentle aesthetic.
- Background color: A very light, almost white tone with a subtle pink tint (#FAF9FA) for an airy and clean canvas.
- Accent color: A calming sage green (#9DC18B) used for interactive elements and subtle highlights, providing a harmonious contrast.
- Body and headline font: 'PT Sans' (sans-serif) for its modern yet friendly and readable characteristics, aligning with a cute and simple design.
- Use rounded, line-based icons for playback controls and navigation, maintaining a simple and approachable visual style.
- A clean and intuitive two-column layout with album art prominently displayed on one side, and the tracklist with 'now playing' details on the other. Player controls are fixed at the bottom.
- Subtle and smooth fade-in/fade-out animations for track transitions and interface updates, contributing to a fluid and cute user experience.
